Features

  • Sistrix Visibility Index for the domain of the currently loaded url is gotten live and displayed in the badge below the extension icon on the top right in the browser's extension bar.
  • Sistrix metric pages for domain, path (up to three directory paths, if available) and url with a single click - left click on the extension icon opens link list with metric pages.
  • All Sistrix metric pages are available with any of 30 countries Sistrix offers. The country can be set by user, on this page below in the dropdown and anytime later on the extension's options page).

Please note!

This extension works only if:

  • You have a Sistrix account
  • You are successfully logged in into Sistrix account
  • You haven't forgotten to set the country for querying Sistrix in the dropdown below

Setup steps you must accomplish to get this extension work:

Step 1: Country setup Step 2: Sistrix login

Your country setting will act like a subdomain in Sistrix urls like https://XX.sistrix.com/example.com

Open Sistrix login page

Don't forget always to login into Sistrix before use this extension.
